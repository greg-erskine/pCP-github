Jivelite

Jivelite is a cut down Squeezebox control application.

piCorePlayer fully supports Jivelite on the Raspberry Pi Foundation's 7" Touch Screen.

Installation: Jivelite can be installed on piCorePlayer via the [Tweaks] page on the piCorePlayer web interface.

Orginally developed and released by Adrian Smith.

Currently maintained by Ralph Irving.
