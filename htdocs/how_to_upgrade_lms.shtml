How-to upgrade Logitech Media Server on your piCorePlayer to 7.9.1

It's easy to install Logitech Media Server on your piCorePlayer using its settings menus. As of this writing piCorePlayer is installing Logitech Media Server 7.9.0. If you want or have to install the 7.9.1 nightly build, then you can follow these instructions.

Should 7.9.1 be outdated by the time you read these instructions, just use the new version number instead
Step-by-step instructions

1.

Access piCorePlayer via ssh—see How-to access piCoreplayer via ssh.

2.

$ cd /tmp

3.

$ wget https://raw.githubusercontent.com/paul-1/lms-update-script/Master/lms-update.sh

4.

$ chmod 755 lms-update.sh

5.

$ sudo ./lms-update.sh --mm 7.9.1 -s -r -u
