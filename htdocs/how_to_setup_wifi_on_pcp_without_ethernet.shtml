How-to setup wifi on piCorePlayer without Ethernet

There are 2 situations where these methods will be useful:

  1. You have a Raspberry Pi without in-built ethernet.
  2. Wired ethernet is not available.
Add "newconfig.cfg" to boot partition on PC

1.

Burn image to a SD card—see How-to burn piCorePlayer onto a SD card.

Note: Do not eject the SD card as you need to add "newconfig.cfg" to the boot partition.

2.

Type the following using your text editor (ie. Notepad++).

3.

Save As the file "F\:newconfig.cfg" to the boot partition.

On a PC this will be a drive letter. (ie. F:)

4.

Eject SD card.

5.

Insert SD card into piCorePlayer.

6.

Apply power.
Using setup with keyboard and monitor

1.

Login into your piCorePlayer.

login as: tc

password: piCore (default)

2.

Type "setup".

3.

Click [Continue].

4.

Click [OK].

5.

Click [No].

6.

Click [On].

7.

Enter SSID.

Enter Password.

Enter Security.

Click [OK].

8.

Click [Yes].

9.

Click [Exit].

10.

Type "pcp br".
