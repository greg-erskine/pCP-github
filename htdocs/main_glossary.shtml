Glossary

ALSAequal

An Equalisier that works with ALSA—see the C* Audio Plugin Suite.

jivelite

Touch screen graphical user interface that works like a Logitech Touch.

LMS

Logitech Media Server (formally Slimserver, SqueezeCentre and Squeezebox Server).

pCP

piCorePlayer.

piCore

A version of TinyCore for the Raspberry Pi.

piCorePlayer

Squeezebox replacement using Raspberry Pi. piCore + squeezelite. Many more options.

putty

Software to connect to your piCorePlayer in terminal mode.

Raspberry Pi

Cheap credit card sized computer that makes an excellent basis for a streaming music player—see Raspberry Pi Foundation.

squeezelite

Software that makes your computer work like a Logitech Squeezebox.

ssh

Software to connect to your piCorePlayer in terminal mode.
