Wifi does not work

A list of common wifi issues in setting up piCorePlayer.

1.

Wrong password.

2.

Wrong SSID.

3.

piCorePlayer does not support hidden SSID.

4.

piCorePlayer does not support special characters in SSID or password.

5.

Wifi adaptor does not support wifi channel selected on wifi router.

6.

USB wifi adaptor and buit-in wifi adaptor both trying to run at the same time.

7.

Some audio cards interfere with Raspberry Pi built-in wifi adaptors.

8.

Wifi drivers gradually changing from wext to nl80211 standard.

9.

Occassionally, kernel upgrades break USB wifi drivers. Unfortunately, we can not test all USB wifi drivers prior to each piCorePlayer release. Please report any issue.
