Getting started - Getting piCorePlayer up and running on your Raspberry Pi.

1.

Download piCorePlayer—see How-to download piCorePlayer.

2.

Burn image to a SD card (ie. using Win32Diskimager).

3.

Insert the SD card, a LAN cable into Raspberry Pi and apply power.

4.

Out of the box, the analog audio via the 3.5mm jack will be working.

5.

piCorePlayer can be controlled like any other Squeezebox player, via:

 
  • Logitech Media Server (LMS) web interface
  • software on your smartphone (ie. iPeng)
  • Duet Controller
  • Squeezebox Radio
  • Squeezebox Touch
