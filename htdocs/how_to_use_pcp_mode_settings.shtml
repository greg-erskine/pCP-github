How-to use piCorePlayer mode settings

piCorePlayer has a concept of Modes. The idea of "mode of operation" is to make the web interface less complicated for the inital setup and especially for new users. Initially, the Mode tabs are only on the bottom left hand side of the [Main Page] and none are highlighted.

When you first install piCorePlayer, it is in "Initial" mode which contains the minimum settings required to make piCorePlayer work.

As you get more experienced with the web interface you can increase the level of complication by setting more advanced modes.

piCorePlayer Mode Settings

Mode

Description

Initial

This mode shows only 12 basic piCorePlayer operations. This should give the user just enough settings to get piCorePlayer working.

Basic

Adds a few more options to [Squeezelite Settings] and [Tweaks] pages. Proabably the minimal mode of usage.

Normal

Adds a more options to [Main Page], [Squeezelite Settings], [Tweaks] and [About] pages.

[LMS] page is now activated.

[Mode] tabs added to the bottom left hand corner of [Squeezelite Settings], [Tweaks] and [LMS] pages.

[piCorePlayers] tabs added to the top left hand side of [Main], [Squeezelite Settings], [Tweaks], [LMS] and [About] pages.

As the name implies, this is considered a normal mode of usage.

Advanced

This the recommended full piCorePlayer system for advanced users.

[Simple Controls] tabs added below [piCorePlayers] tabs on the [Main], [Squeezelite Settings], [Tweaks], [LMS] and [About] pages.

Beta

This adds options that are considered to still be in beta. These options may not work properly, are still under development or will possibly break between versions. Some beta options have been added by the developers for their presonal use but may be useful to the more advanced user or hacker.

As beta options prove useful and reliable, they will be normally be moved to Normal or Advanced mode.
