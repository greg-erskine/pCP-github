How-to access piCorePlayer via ssh

1.

Start your ssh client software (ie. PuTTY).

2.

Enter your IP address.

3.

Set Port to 22.

4.

Select SSH.

5.

Click [Open].

6.

Login into your piCorePlayer.

login as: tc

password: piCore (default)

7.

Type "exit" or click [X] to close terminal.
