How-to edit config.txt

config.txt is found on piCore's boot partition /mnt/mmcblk0p1

By default, after booting, the boot partition is unmounted.

So, to edit config.txt, you must first mount the boot partition, then edit using vi.

Note: You must reboot piCorePlayer after changing config.txt to activate changes.
Shorthand method—using pCP aliases

1.

Access piCorePlayer via ssh—see How-to access piCoreplayer via ssh.

2.

$ m1

3.

$ c1

4.

$ vicfg

5.

$ pcp rb
Longhand method—using standard Linux commands

1.

Access piCorePlayer via ssh—see How-to access piCoreplayer via ssh.

2.

$ mount /mnt/mmcblk0p1

3.

$ cd /mnt/mmcblk0p1

4.

$ vi config.txt

5.

$ reboot
